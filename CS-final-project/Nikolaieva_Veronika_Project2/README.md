# README

This folder contains `analyzer.py`, which takes the name of a file containing sequences to analyze.

## How to use:

In the terminal, type:
`python analyzer.py`

Press Enter.

When prompted:
`Enter filename:`
type the name of your file and press Enter.

Example:
`Enter filename: edge_cases_test.txt`

## Testing:

`analyzer.py` was tested using the file `edge_cases_test.txt`.
